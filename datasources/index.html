Global data sources —
where NAITRO's numbers come from.

Every score, threshold and recommendation on this platform traces back to a named source. This page lists the datasets, APIs and frameworks NAITRO draws on — what's live and queryable today, and what's used as a reference framework.

Continent-wide, 30m-resolution machine-learning soil property service covering ~30 properties per coordinate (pH, N, P, K, CEC, texture, carbon) with depth stratification and uncertainty bounds. Queried for SA1, SA2, and SA3.

Reference frameworks
Scientific and policy frameworks NAITRO's scoring is built on — no direct API integration
IFA / fertilizer.org
International Fertilizer Association
Reference
Global NUE tracking methodology, 4R Nutrient Stewardship framework, and the Scope 3 emissions reduction pathway analysis (with Systemiq). Basis for the NUE science section in How NAITRO Works.
NUE 4R Stewardship Policy
fertilizer.org
FAOSTAT
UN Food and Agriculture Organization
Reference
Country-level cropland nutrient use efficiency trends, cited as the data source behind the global 55% NUE figure used throughout the platform.
National NUE Country data
fao.org/faostat
Open Bodemindex (OBI)
Wageningen University Research, with Rabobank, a.s.r., Vitens
Reference
21-indicator open-source soil quality scoring framework across Chemical, Physical, Biological, and Management domains. The architecture NAITRO's farm-level Soil Health Score is adapted from for NZ and SA conditions.
Scoring framework Netherlands
No public API
NPS-FM 2020 / Overseer
NZ Ministry for the Environment
Reference
New Zealand's freshwater nutrient compliance regulation. OverseerFM, the prior modelling tool used to assess compliance, was found "not fit for purpose" by the Office of the Chief Science Advisor in 2021 — the regulatory gap NAITRO's IoT-verified scoring is designed to fill.
